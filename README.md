# PhonePe Pulse Dashboard 📊

An interactive **Power BI dashboard project** built using the **PhonePe Pulse dataset** to analyze digital payment trends, user growth, geo-level performance, transaction behavior, and brand-level user insights across India.


## 🚀 Project Overview

The **PhonePe Pulse Dashboard** was created to transform raw transaction and user data into a **business-focused analytics solution**.
The dashboard helps answer key questions such as:

* How large is the PhonePe transaction ecosystem?
* Which states and districts contribute the most to transactions and users?
* How is PhonePe growing over time?
* Which transaction categories drive the most value and usage?
* Which smartphone brands dominate the user base?

The project was built in **Power BI** with multiple dashboard pages, interactive slicers, KPI cards, geo-visualization, trend analysis, and DAX-based business measures.

## 🎯 Objectives

* Build an interactive dashboard using the **PhonePe Pulse dataset**
* Analyze **transaction amount, transaction count, and registered users**
* Create **geo-based insights** across Indian states and districts
* Identify **top-performing states, districts, and transaction categories**
* Track **growth trends using YoY KPIs**
* Explore **brand-level user distribution**
* Present insights in a business-friendly dashboard format


## 🛠️ Tools & Technologies Used

* **Power BI**
* **DAX (Data Analysis Expressions)**
* **Data Modeling**
* **Power Query / Data Cleaning**
* **PhonePe Pulse Dataset**


## 📂 Dashboard Structure

The dashboard is divided into **4 analytical pages**:

### **1. Executive Overview Dashboard**

Provides a high-level summary of PhonePe’s performance through:

* Total Transaction Amount
* Total Transaction Count
* Average Transaction Value
* Registered Users
* Transaction trend over time
* Top states by transaction amount
* Dynamic metric comparison

### **2. Geo Analysis Dashboard**

Focuses on geographic performance and regional spread through:

* India geo map
* Top states by selected geo metric
* Top districts by selected geo metric
* Geo trend analysis over time

### **3. Growth & Top User Insights Dashboard**

Analyzes platform growth and top-performing markets using:

* Top transaction amount
* Top transaction count
* Top registered users
* Latest YoY transaction growth
* Top 10 states by amount, count, and users
* Growth trend line

### **4. Transaction & User Segment Analysis Dashboard**

Explores transaction categories and brand-level user behavior using:

* Transaction type by amount
* Transaction type by count
* Top brands by user count
* Transaction category trend over time
* YoY registered user growth


## 📌 Key Features

* **Interactive slicers** for YearQuarter, State, and Metric selection
* **Dynamic KPI cards** for business performance tracking
* **Geo map visualization** for state-level and district-level insights
* **Top N analysis** for states, districts, and brands
* **Year-over-Year growth KPIs**
* **Transaction category comparison**
* **Brand-wise user distribution analysis**


## 📊 Key Business Insights

* **Peer-to-peer payments** contribute the highest transaction value, showing strong dominance in direct digital transfers.
* **Merchant payments** lead in transaction count, indicating frequent day-to-day usage.
* States like **Telangana, Maharashtra, and Karnataka** consistently perform strongly across transaction and user metrics.
* The dashboard shows a **steady upward trend in transaction amount**, indicating strong platform growth over time.
* **YoY growth in transactions and users** highlights continued expansion of the PhonePe ecosystem.
* Major smartphone brands such as **Xiaomi, Samsung, and Vivo** contribute significantly to the user base.
